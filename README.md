# AKR1B1-Gastric-Cancer-PCD-Prognosis
Deciphering Programmed Cell Death for Gastric Cancer Prognosis: AKR1B1 as a Key Player
## Project Description
Bioinformatics analysis code for exploring programmed cell death (PCD) in gastric cancer (GC) and validating AKR1B1 as a key prognostic biomarker.
## Required R Packages
survival, survminer, glmnet, randomForest, e1071, sigFeature, ggplot2, timeROC, GSVA, estimate, Seurat, MAESTRO, pROC, boot
## Usage
1. Clone this repository
2. Install required packages
3. Run the main analysis script
## Key Content
Data preprocessing, Cox regression, LASSO model, machine learning for gene selection, immune infiltration, drug sensitivity and single-cell analysis.
